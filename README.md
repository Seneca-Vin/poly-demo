**本demo基于springboot ，jdk1.8 构建，演示菠萝派自建商城的签名验证和一部分官方文档实体类封装**

## 使用步骤：

1、配置好jdk1.8，maven后即可启动使用

2、需要提前拿到自己的key，token，appSecret参数

3、对接讲解等可以看[博客](https://blog.csdn.net/qq467063434/article/details/132857757)
